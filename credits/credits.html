What's new?

Fixed the calculation under Interval mode when you spin the Playback picker.

Interval Calculations - You can now switch the calculator into Interval mode by toggling the switch above. This allows you to calculate the Interval (in seconds) between photos to produce a timelapse of a set length.

To quickly switch between interval mode and normal mode in the calculations screen, double-tap the Interval label. A red dot will appear next to the Inveral label when it is in Interval Calculation mode.
